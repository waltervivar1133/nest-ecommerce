# Teslo API

1.- Clonar proyecto 
2.-  ```yarn install ```
3 .- Clonar archivo ```env.template```  a  ```  .env ```
4.- Cambiar las variables de entorno 
5.- Levantar la base de datos

 ```
 docker-compose  up  -d 
 ```

 6.- Levantar: ``` yarn start:dev ```
